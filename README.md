## Personal Finance Tracker
A simple command-line Personal Finance Tracker built in Python as part of my learning journey in Python, Git, and GitHub.

About the Project
This application allows users to manage their income and expenses through a menu-driven interface. It calculates the current balance and allows users to continue using the menu until they choose to exit.

Features
- Add Income
- Add Expense
- View Current Balance
- View transaction history
- Menu-driven interface
- Continue or Exit option
- Input validation for menu choices
