## Personal Finance Tracker
A desktop-based Personal Finance Tracker built with Python and Tkinter. The application helps users record income and expenses, maintain transaction history, visualize financial data, and automatically save transactions using file handling.

## About the Project
This application allows users to manage their income and expenses through a menu-driven interface. It calculates the current balance and allows users to continue using the menu until they choose to exit.

## Features
- Add Income
- Add Expense
- View Current Balance
- View transaction history
- Financial Summary
- Clear Transaction History
- Automatic Balance Recovery 
- Persistent file storage
- Graphical Representation
- GUI using Tkinter
- Input validation
